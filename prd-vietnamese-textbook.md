## 1. Executive Summary

为母语为汉语、零基础或少量基础的中国人，写一份中越双语、有声配套的 30 节速成越南语教材。

教材分五个阶段：**发音 → 日常会话 → 语法骨架 → 阅读写作 → 文化语境**。每一节都针对汉语母语者做两件事：用对比图讲清越南语独有的声调和元音，把源自古汉语的 6 成词汇变成速记优势。

学完 30 节，学员的越南语水平达到 **CEFR A1**，能独立完成越南旅行和日常初阶沟通。

**为什么现在做**：中越贸易、旅游、跨境务工的需求在涨，但中文母语者能用的越南语教材很少（🔶 具体数据待补、🔶 市场调研待做）。

## 2. Problem Statement

### 谁有这个问题？
- 🔵 核心学员：25-45 岁中国人，因为工作、商务、旅游、婚恋、学术等具体动机想学越南语
- 🔵 次要学员：在华越南语专业本科生、跨境业务从业者子女

### 问题是什么？
中国人学越南语会遇到四类结构性困难：

1. **声调体系陌生**：越南语有 6 个声调（平、玄、问、跌、锐、重），跟汉语的 4 声调 + 轻声不完全重合
2. **元音辅音差异**：ư / ơ / â / ă 这几个音在汉语里没有对应音位，发音矫正要花大量时间
3. **汉字词识别盲区**：60-70% 的越南语词汇源自古汉语，对中国人其实是捷径，但多数教材没把这个优势用起来
4. **语法陷阱**：定语后置、SVO 语序、大量虚词（đã / sẽ / đang / được / rồi）、自成一套的量词

### 痛在哪里？
- 🔶 现有越南语教材大多面向越南人学中文，或者面向英语母语者，没把汉语母语者的发音优势和汉字词捷径当成核心设计点
- 🔶 缺少科学分层：从入门到 B1 没有明确的能力目标、阶段验收、配套练习
- 学习者花 200+ 小时还是卡在"能认不能讲、会写不会听"

### Evidence
- 🔵 待补：3-5 条目标学员访谈、痛点调研

## 3. Target Users & Personas

### 主力用户：商务赴越的张经理
- **年龄 / 角色**：35 岁，中国制造业经理，常驻胡志明市
- **学习动机**：跟越方供应商沟通、看合同、参加晚宴
- **痛点**：发音不准被笑、商务敬语不会用
- **目标**：6 个月内达到 A2，能独立开会
- **学习时间**：工作日每晚 30-45 分钟，周末 2 小时

- **年龄 / 角色**：28 岁，旅行者，准备越南 10 日深度游
- **学习动机**：不想只跟团，要自己点菜、砍价、问路
- **痛点**：短期记不住
- **目标**：3 周速成生存口语
- **学习时间**：每天通勤 20 分钟

### 第三类用户：越南语专业本科新生
- **学习动机**：零基础系统学习，4 年后达到 B2-C1
- **痛点**：需要正规教材 + 教辅
- **目标**：大一结束时达到 A2

## 5. Solution Overview

### 5.1 教材定位
面向汉语母语者、30 节速成版的越南语教材，学完达到 CEFR A1。

### 5.2 教材结构

| 阶段 | 名称 | 节数 | 能力目标 |
|---|---|:---:|---|
| **S1 启航** | 语音 + 字母 + 打招呼 | 6 | — |
| **S2 入门** | 日常会话（点餐、问路、购物） | 8 | A1.0 |
| **S3 进阶** | 语法骨架（时态、虚词、句子） | 8 | A1.5 |
| **S4 实战** | 听力 + 写作 + 文化情境 | 5 | A1.8 |
| **S5 收束** | 综合自测 + 旅行 / 商务选修 | 3 | A1 → A2 入口 |

### 5.3 配套音频
- ✅ 每节配课文、例句、词汇的标准发音（河内音）
- 🔵 待定：是否加慢速版供跟读
- 🔵 待定：是否做音频 App 或小程序扫码听

### 5.4 每章标配结构（8 件套）
1. 学习目标（SMART，可验收）
2. 发音提示（🔊 对比汉语）
3. 核心词汇（标汉字词 + 词源）
4. 语法点（含汉语母语者易错预警 ⚠️）
5. 对话 / 课文（含文化注释）
6. 练习（听说读写译）
7. 阶段自测（达标才能进下一章）
8. 拓展（汉字词文化、越南小知识）

### 5.5 差异化特色
- **汉字词专栏**：每章标记源自汉语的词，配词源解释
- **汉越发音对比图**：声调、元音、辅音并排
- **易错预警**：基于汉语母语者典型偏误数据库
- **文化侧栏**：越南礼仪、饮食、节庆

## 7. User Stories & Requirements

### Epic
作为汉语母语者的越南语初学者，我想要一份针对我的发音习惯、汉字词优势和典型偏误专门设计的教材，让我用更少时间达到可用的越南语水平。

### User Stories（示例）

**US-01：认识越南语字母表**
> 作为零基础学员，我能在 2 课时内认全 29 个字母 + 6 个声调符号，并正确书写。

**US-02：掌握汉越发音对应**
> 作为有汉语基础的学员，我能借助对比表快速学会 ư / ơ / â 等新元音。

**US-03：识别汉字词**
> 作为懂汉字的学员，我能识别 60% 的基础词汇（因为它们源自古汉语）。

**US-04：日常对话**
> 作为旅客，我能在 S2 结束后独立完成点餐、问路、砍价。

### Acceptance Criteria（US-01 示例）
- [ ] 列出全部 29 字母 + 6 声调
- [ ] 听音频能写出听到的字母
- [ ] 自测通过率 ≥ 90%

## 7b. 30 节章节大纲（核心交付物）

> 每节结构：学习目标 / 发音提示 / 词汇（含汉字词标注）/ 语法点 / 对话 / 练习 / 文化侧栏 / 自测

### S1 启航：语音 + 字母 + 打招呼（6 节）

| # | 章节 | 学习目标 |
|---|---|---|
| 1 | 越南语简介 + 字母表 | 认全 29 字母（元音 + 辅音 + 9 个二合字母） |
| 2 | 6 个声调 + 调号书写 | 读写 6 个声调符号，区分平 / 玄 / 问 / 跌 / 锐 / 重 |
| 3 | 汉越元音对比（ư / ơ / â / ă） | 掌握汉语没有的 4 个特殊元音 |
| 4 | 汉越辅音对比（ch / tr / nh / ng） | 区分易混辅音对，能正确发音 |
| 5 | 打招呼 + 自我介绍 | 用"你好 / 我叫… / 来自… / 很高兴认识你" |
| 6 | S1 自测 + 数字 1-100 | 巩固声调 + 字母，能听懂、说出 1-100 |

### S2 入门：日常会话骨架（8 节）

| # | 章节 | 学习目标 |
|---|---|---|
| 7 | 家庭称谓 + 年龄 | 介绍家庭成员、问年龄 |
| 8 | 时间 + 日期 + 星期 | 表达"今天几号 / 现在几点 / 周几" |
| 9 | 天气 + 季节 | 描述天气、季节性活动 |
| 10 | 问路 + 方位 | 用"在…前面 / 旁边 / 对面"问路 |
| 11 | 点餐 + 餐厅用语 | 完整点餐对话、看懂菜单 |
| 12 | 购物 + 砍价 | 问价、砍价、付钱、找零 |
| 13 | 酒店入住 + 电话 | 订房、入住、打电话常用句 |
| 14 | S2 自测 + 旅行高频 100 句 | 模拟旅行 10 大场景 |

### S3 进阶：语法骨架（8 节）

| # | 章节 | 学习目标 |
|---|---|---|
| 15 | 名词 + 量词（与汉语对比） | 掌握 15 个核心量词 |
| 16 | 代词 + 指示词（này / đó / kia） | 区分"这 / 那 / 那边" |
| 17 | 动词 + 时态（đã / đang / sẽ） | 过去 / 正在 / 将来三时态 |
| 18 | 形容词 + 比较（A hơn B / A nhất） | 越南语比较级与最高级 |
| 19 | 否定 + 疑问句式 | 4 种否定 + 5 种问句 |
| 20 | 介词 + 方位短语 | trong / ngoài / trên / dưới / trước / sau |
| 21 | 连接词 + 复合句 | vì…nên / nếu…thì / tuy…nhưng |
| 22 | S3 自测 + 综合语法演练 | 写 100 字短文 |

## 8. Out of Scope

**本教材不做**
- ❌ 古越南语 / 喃字（Chữ Nôm）专章 — 学者向，初学者用不到
- ❌ 越南文学精读（仅做选段作为选修）
- ❌ 全专业术语手册（仅覆盖商务、旅游、学术基础）
- ❌ AI 语音评测系统（本期为 PDF 教材，可后续配套）
- ❌ 同传 / 高阶口译训练

## 9. Dependencies & Risks

### Dependencies
- 🔵 越南语原文素材库（对话 / 课文文本 + 音频）
- 🔵 母语审校（避免教材硬伤）

### Risks & Mitigations

| 风险 | 缓解 |
|---|---|
| 学员水平跨度大 | 设分层目标（速成 / 系统 / 专业三档） |
| 越南语素材版权 | 优先原创 + 公版 |
