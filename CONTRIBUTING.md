## 中文

### 我们欢迎什么

| 类型 | 适合谁 | 流程 |
|---|---|---|
| 🐛 **错漏修复** | 任何读者 | 直接提 PR，一句话说明 |
| 🌍 **翻译改进** | 中越双语者 | PR 标 `translation` |
| 🎙️ **音频优化** | 母语者 | 联系邮箱，发 demo |
| 📚 **内容补充** | 教师 / 高阶学员 | 提 Issue 讨论 |
| 💡 **教学法建议** | 越南语教师 | 提 Issue 讨论 |

### 怎么贡献

#### 1. 错漏修复（最快上手）

```bash
# Fork 仓库
# 找到 .md 文件直接改
# 提 PR，标题格式: [fix] 简短描述
# 例如: [fix] 节 5 谐音 "新操" 应为 "辛曹"
```

不需要开 Issue，可以直接 PR。

#### 2. 内容补充

1. 提 Issue，标签 `content-addition`
2. 描述：补什么 / 放哪节 / 内容草稿
3. 等作者确认后再写

#### 3. 翻译改进

如果你的中越双语表达更地道：

1. 找到对应 .md 文件
2. 改完后跑一遍：中文读一遍 / 越南语读一遍
3. 提 PR，标签 `translation`

#### 4. 音频优化

我们目前用 coco（河内人）的录音。如果你有更标准的河内音：

1. 录 5 分钟 demo（任意一节任意一段）
2. 邮件发到 tidy-vietnamese@example.com
3. 主题：`audio demo - 你的名字`

### 内容规范

#### 章节结构（8 件套）

每节文档**必须**包含这 8 段（顺序固定）：

1. `## 1. 学习目标` — SMART，可验收
2. `## 2. 发音提示` — 含汉越对比
3. `## 3. 核心词汇` — 表格 + 汉字词标注
4. `## 4. 语法点` — 含易错预警
5. `## 5. 对话 / 课文` — 越南语 + 中文翻译
6. `## 6. 练习` — 听说读写译
7. `## 7. 自测` — 5-10 题
8. `## 8. 文化侧栏` — 越南文化 / 易错陷阱

#### 越南语排版

- 越南语用**斜体**或 `code` 块，与中文区分
- 汉字词用**粗体**：**gia đình**（家庭）
- 谐音用括号：Xin chào（新 操）

#### 易错预警标记

用 `⚠️` 开头：

```markdown
⚠️ 易错预警：越南人**很在意称谓**，叫错年龄辈分对方会觉得你没礼貌。
```

## English

### We welcome

| Type | For | Flow |
|---|---|---|
| 🐛 **Error fixes** | Any reader | Direct PR |
| 🌍 **Translation** | Bilingual speakers | PR with `translation` label |
| 🎙️ **Audio** | Native speakers | Email demo |
| 📚 **Content** | Teachers / advanced | Open Issue |
| 💡 **Pedagogy** | Vietnamese teachers | Open Issue |

### How to contribute

#### 1. Error fixes (fastest)

```bash
# Fork the repo
# Edit .md files directly
# Open PR: [fix] short description
# Example: [fix] Lesson 5 phonetic "新操" should be "辛曹"
```

No issue needed — direct PR is fine.

#### 2. Content additions

1. Open Issue with `content-addition` label
2. Describe: what / which lesson / draft
3. Wait for author confirmation before writing

#### 3. Translation improvements

If your Chinese-Vietnamese bilingual is more idiomatic:

1. Find the .md file
2. Read through: Chinese once / Vietnamese once
3. Open PR with `translation` label

#### 4. Audio improvements

Currently we use coco's recordings. If you have more standard Hanoian:

1. Record 5-min demo (any lesson, any section)
2. Email to tidy-vietnamese@example.com
3. Subject: `audio demo - your name`

### Content standards

#### 8-section lesson format

Every lesson **must** have these 8 sections in this order:

1. `## 1. Learning Objectives` — SMART, verifiable
2. `## 2. Pronunciation` — Chinese-Vietnamese comparison
3. `## 3. Vocabulary` — table with Sino-Vietnamese labels
4. `## 4. Grammar` — with error warnings
5. `## 5. Dialogues` — Vietnamese + Chinese translation
6. `## 6. Exercises` — listening / speaking / reading / writing / translation
7. `## 7. Self-test` — 5-10 questions
8. `## 8. Culture` — Vietnamese culture / pitfalls

#### Vietnamese typography

- Vietnamese in *italic* or `code` blocks to distinguish from Chinese
- Sino-Vietnamese in **bold**: **gia đình** (family)
- Phonetic in parentheses: Xin chào (xin chao)

#### Error warning marker

Start with `⚠️`:

```markdown
⚠️ Warning: Vietnamese are sensitive to address terms; wrong age/hierarchy feels rude.
```
