## 中文

### 这是什么

`tidy-vietnamese` 是一份**开源的越南语教材**，作者 coco 是越南河内人。这本教材的目标是：**让母语为汉语的中国人，用最少时间学到能用的越南语**。

我们做这件事的核心理由是：**中国人学越南语其实有天然优势**——60% 的越南语词汇源自古汉语（"汉字词"），你看到 **gia đình**（家庭）、**học sinh**（学生）、**kinh tế**（经济）这些词，几乎能猜到意思。但市面教材没把这个优势用起来，多是为英语母语者设计的。

这本教材专为中国人设计。

### 特点

- 🎯 **针对汉语母语者**：每节都有"汉越对比"发音、汉字词速记、易错预警
- 🆓 **完全免费**：PDF + 配套音频全部开源
- 🗣️ **配套标准河内音音频**：每节配课文 / 例句 / 词汇标准发音
- 📚 **结构清晰**：30 节分 5 阶段（启航 / 入门 / 进阶 / 实战 / 收束），学完达 CEFR A1
- 🔀 **中越双语**：中文讲解 + 越南语原文
- 📖 **8 件套章节结构**：每节含学习目标 / 发音 / 词汇 / 语法 / 对话 / 练习 / 自测 / 文化侧栏

### 学习路径（30 节 · 4-6 周）

```
S1 启航 (节 1-6)        →  语音 + 字母 + 打招呼
   ↓
S2 入门 (节 7-14)       →  日常会话（点餐 / 问路 / 购物 / 旅行）
   ↓
S3 进阶 (节 15-22)      →  语法骨架（时态 / 虚词 / 句子 / 比较）
   ↓
S4 实战 (节 23-27)      →  听力 / 写作 / 文化（礼仪 / 饮食 / 商务）
   ↓
S5 收束 (节 28-30)      →  旅行 5 日模拟 / 商务寒暄 / 毕业自测
```

学完 30 节，你的越南语水平达到 **CEFR A1**，能独立完成越南旅行和日常初阶沟通。

### 怎么用

1. **按顺序学**：从 [第 1 节 字母表](lessons/01-gioi-thieu-bang-chu-cai.md) 开始
2. **听音频**：每节配 `audio/lesson-XX/` 标准河内音
3. **做完自测**：每节有 5-10 题自测，通过才能进下一节
4. **反馈问题**：发现错漏 → 提 Issue / 微信群 / 邮箱（见下方反馈入口）

## English

### What is this?

`tidy-vietnamese` is a **free, open-source Vietnamese textbook** designed specifically for **native Mandarin Chinese speakers**. The author, coco, is a native Hanoian Vietnamese speaker.

The core insight: **Chinese speakers have a natural advantage learning Vietnamese** — 60% of Vietnamese vocabulary derives from Classical Chinese ("Sino-Vietnamese" or *từ Hán Việt*). When you see **gia đình** (family), **học sinh** (student), **kinh tế** (economy), you can almost guess the meaning. But most textbooks don't leverage this — they're designed for English speakers.

This textbook is designed for Chinese speakers.

### Features

- 🎯 **Mandarin-focused**: every lesson has Chinese-Vietnamese pronunciation comparison, Sino-Vietnamese vocabulary tips, and common error warnings
- 🆓 **Completely free**: PDF + audio all open-source
- 🗣️ **Standard Hanoian audio**: every lesson includes standard Hanoi pronunciation
- 📚 **Clear structure**: 30 lessons in 5 stages, target CEFR A1
- 🔀 **Bilingual**: Chinese explanations + Vietnamese source
- 📖 **8-section lesson format**: objectives / pronunciation / vocabulary / grammar / dialogues / exercises / self-test / culture

### Learning Path (30 lessons · 4-6 weeks)

```
Stage 1 Launch (1-6)        →  Alphabet + tones + greetings
   ↓
Stage 2 Foundation (7-14)   →  Daily conversation (ordering, directions, shopping, travel)
   ↓
Stage 3 Intermediate (15-22) →  Grammar backbone (tenses, function words, comparisons)
   ↓
Stage 4 Practical (23-27)   →  Listening / writing / culture (etiquette, food, business)
   ↓
Stage 5 Capstone (28-30)    →  5-day travel sim / business greetings / final test
```

After 30 lessons, your Vietnamese reaches **CEFR A1** — enough for travel and basic daily conversation.

### How to use

1. **Sequential learning**: start with [Lesson 1: Alphabet](lessons/01-bang-chu-cai.md)
2. **Listen to audio**: every lesson has `audio/lesson-XX/` standard Hanoi pronunciation
3. **Self-test**: 5-10 questions per lesson — pass to proceed
4. **Report issues**: found an error? Open an Issue / join WeChat group / email (see below)
